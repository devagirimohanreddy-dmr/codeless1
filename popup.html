Robot Framework Test Builder

How to build your test:

  1. Click Start Inspection and select an element on the page
  2. Enter a descriptive Element Name (e.g., "Login Button")
  3. Choose a Locator Type (XPath, ID, etc.)
  4. If using XPath, select from available XPath options
  5. Choose an Action to perform on the element
  6. Add any required Input Values
  7. Click Add Step to include it in your test
  8. Repeat for additional steps
  9. Click Generate Robot Script when done
Scan elements and add actions to build your test script.
